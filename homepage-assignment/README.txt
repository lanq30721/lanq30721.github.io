个人主页使用说明

当前网站目录：
- index.html
- style.css
- script.js
- assets/profile-photo.jpg
- assets/research-wave.svg

上传到 GitHub Pages 时，请把整个目录中的文件一起上传。

推荐仓库名：
你的 GitHub 用户名.github.io

GitHub Pages 设置：
1. 打开仓库 Settings
2. 点击 Pages
3. Branch 选择 main
4. Folder 选择 / (root)
5. 保存后等待几分钟

上线地址：
https://你的 GitHub 用户名.github.io
